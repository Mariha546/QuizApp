This project is an AI-powered Adaptive Quiz Generation System developed using Java and Swing. It enables users to register, log in, and generate quizzes based on selected topics. The system uses an AI engine to create multiple-choice and true/false questions, while features such as automatic quiz saving, score evaluation, recommendations, and timer management improve the learning experience. Its layered architecture (GUI, Services, Models, and Utilities) promotes modularity, maintainability, and efficient data management.
